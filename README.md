# DB-CAUNet: Dual-Branch Cross-Attention U-Net for Lung Nodule Segmentation

## Overview

DB-CAUNet (Dual-Branch Cross-Attention U-Net) is a deep learning framework developed for automated lung nodule segmentation from chest CT images. The model extends the conventional U-Net architecture by introducing a dual-branch cross-attention mechanism that enhances encoder-decoder feature interaction and improves contextual feature learning.

The framework is designed to address challenges commonly encountered in lung nodule segmentation, including:

- Low contrast between nodules and surrounding tissues
- Irregular nodule boundaries
- Variations in nodule size and shape
- Complex background structures
- Class imbalance in medical image segmentation

## Features

- Dual-Branch Cross-Attention Module
- U-Net-based Encoder–Decoder Architecture
- CT Image Preprocessing Pipeline
- Data Augmentation Support
- Training and Validation Framework
- Segmentation Inference Pipeline
- Performance Evaluation Metrics
- Modular and Reproducible Implementation
